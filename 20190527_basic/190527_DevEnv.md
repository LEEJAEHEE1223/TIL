## 2. CLI - 터미널 명령어 학습

windows에서 gitbash 검색후 실행

```sh
$ pwd       # 현재 디렉토리 확인
$ cd 폴더명  # 하위(폴더명) 디렉토리로 이동
$ cd ..     # 상위 디렉토리로 이동
$ cd ~      # 홈 디렉토리로 이동
$ cd -      # 뒤로가기
$ mkdir 폴더명 # 새로운 폴더를 생성한다.
$ touch a.txt b.csv  # 2개 파일을 생성한다.
$ ls        # List 현재 폴더 안의 파일/디렉토리 목록을 표시한다.
```
