## MTV 란? (MVC 모델의 다른표현)

#### 모델=model(데이터관리) , 템플릿=view(사용자 화면 html), 뷰=controller(중간관리자 - app.py)

1) 요청순서 request -> View -> Model <-> Database

2) 응답순서 Model -> View -> Template(html) -> response
